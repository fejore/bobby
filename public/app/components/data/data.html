Import

Note : L'import détruira tous les items actuellement associés à votre association pour les remplacer par ceux présents dans vote fichier. Faites donc attention à avoir votre inventaire complet sur ce dernier.

Pour que l'import se passe dans les meilleures conditions un certain nombre de règles doivt être respecté.

Règle générale

Le fichier doit être au format CSV.

Colonnes

Le fichier ne doit pas contenir le nom des colonnes. Dans l'ordre il faut remplir les champs suivants :
Nom / Quantité / Statut / Caution / Catégorie / Localisation

Catégorie

Le champ Catégorie doit être rempli par un des termes suivants (à la majuscule près !) :
{{type.name}} /
Si tu as fait d'une manière différente, Ctrl+H est ton ami ;)

Localisation

Le champ Localisation doit être rempli par un des termes suivants (à la majuscule près !) :
{{place.name}} /

Statut

Le champ Statut doit être rempli par un des termes suivants (à la majuscule près !) :
Visible / Visible et non empruntable / Invisible
